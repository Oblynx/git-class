Σε ποιους απευθύνεται

Μαθητές, φοιτητές, επαγγελματίες - όλοι ευπρόσδεκτοι! Το σεμινάριο είναι δωρεάν.

Θέματα

  • Χρήση git & GitHub για αρχάριους και προχωρημένους
  • Command line, workflows, σωστή χρήση με καθαρό ιστορικό
  • Χρήση ως εργαλείο αναίρεσης και ιστορικού
  • Ατομική χρήση και συνεργασία
  • Συνεργατική χρήση
    • branches
    • forks & pull requests
    • reviews
    • conflicts & merges

Η πρώτη μέρα θα καλύψει τις βασικές έννοιες, ενώ τη δεύτερη θα επεκταθούμε σε πιο προχωρημένες τεχνικές.

Περισσότερες πληροφορίες

  • Όλο το υλικό του σεμιναρίου (διαφάνειες, ασκήσεις, βίντεο, κλπ.) θα δημοσιευθεί σε αυτό το site υπό Creative Commons 3.0 Attribution.
  • Για νέα και ανακοινώσεις, παρακολουθήστε επίσης το event στο Facebook
